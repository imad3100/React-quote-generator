

> A simple react starter application a lightweight alternative to create-react-app, and easly configureable

## Install

```sh
yarn install
```

## Usage

```sh
yarn start
```

## Run tests

```sh
yarn test
```

## Project structure

```sh
my-app-from-scratch/
┣ node_modules/
┣ public/

  ┗ main.js
┣ src/
 ┣ App.j
 ┗ index.js

```
